# Explore NASA

- Astrological Picture of the Day
- Martain Sunset
- Lunar Transit Snap Shots
- Curiosity and Opportunity Images of Mars

#### Link to live site: [Explore NASA](https://explore-nasa.herokuapp.com/)

- Technologies:
    - Javascript
    - AJAX with 4 API calls
    - jQuery
    - CSS
    - HTML

Created an informational web application that allows the user to generate data sets and toggle buttons to open and close the data.

- APOD image toggle button generates a new data set daily.

- Martian Sunset toggle button.

- Lunar transit toggle button. Each time the button is toggled on and off a random image of the image array data set appears. Shows various moon positions in front of the camera.  

- Rover search: Allows user to view random rover images available per specified rover name input. Each time the input is submitted a new image will appear set randomly to the image array's data set.
